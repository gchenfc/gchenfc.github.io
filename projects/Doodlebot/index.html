Doodlebot

Summer, 2016

Intro

Doodlebot was an independent project with the goal of learning the basics of using microcontrollers, motors, and control systems. While there have since been some commercial products which seem to have applied the same core design to more useful applications (Goliath CNC, ZUtA portable printer, Shaper Router), my objective was much more fun/experience based. I wanted to create a simple robot which could draw on any flat surface using a pen. The project was completed start to finish during the summer of 2016 and took around 3 weeks of on/off work. I have lots of ideas for improvements which I'll explain at the end and which maybe I'll get around to someday.
